\paragraph{A short numerical recurrence.}
Let $S_m:=2^{m+1}\EE_{\mathrm{unif}}[G_m]=\sum_{\omega\in\{0,1\}^{m+1}}G_m(\omega)$. Exact enumeration verifies that for all $6\le m\le 22$,
\begin{equation}\label{eq:Sm-recurrence}
S_m = 3S_{m-1}+S_{m-2}-7S_{m-3}+4S_{m-5}.
\end{equation}
The characteristic polynomial factors as
\[
\lambda^5-3\lambda^4-\lambda^3+7\lambda^2-4
=
(\lambda-2)^2(\lambda+1)^2(\lambda-1).
\]
If \eqref{eq:Sm-recurrence} holds for all $m$, the double root at $\lambda=2$ implies $S_m=(a+bm)2^m+O(1)$ and hence $\EE_{\mathrm{unif}}[G_m]=\frac{S_m}{2^{m+1}}=\frac{b}{2}m+O(1)$. The initial values $S_1=1,\dots,S_5=109$ then give $b=\frac{8}{9}$, i.e.\ the candidate slope $\frac{b}{2}=\frac{4}{9}$.
In fact, under \eqref{eq:Sm-recurrence} the unique solution matching the same initial values is
\[
S_m = \Bigl(-\frac{29}{27}+\frac{8m}{9}\Bigr)2^m + \frac54 + \Bigl(-\frac{19}{108}+\frac{m}{18}\Bigr)(-1)^m.
\]
