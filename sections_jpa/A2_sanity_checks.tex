\section{Finite-range checks}\label{app:sanity}

Exact enumeration up to $m=22$ was used only as a consistency check for the finite-window formulas in the main text. Table~\ref{tab:sanity-fibers} lists representative values. Throughout, $F_1=F_2=1$ and $|X_m|=F_{m+2}$. Since the minimum fiber size is always $1$, we record only the maximal fiber size and the second moment
\[
\Sigma_m:=\sum_{x\in X_m} d_m(x)^2.
\]
